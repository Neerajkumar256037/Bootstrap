# 🛒 E-Commerce Website (Bootstrap Project)

This is a simple responsive E-Commerce website built using **Bootstrap 5**, HTML, and CSS.  
It includes basic pages like Home (E-commerce), Login, and Registration.

## 🚀 Features

- Responsive design using Bootstrap
- Simple and clean UI
- 3 main pages:
  - 🛍️ E-commerce (Home page)
  - 🔐 Login page
  - 📝 Registration page
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
